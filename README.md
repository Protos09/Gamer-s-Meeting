# Gamer's-Meeting
 
 
### ◆サービス名
#### Gamer's Meeting

### ◆概要
##### ゲームのレビューを投稿できるwebサービス。<br>採点とコメントをつけることができる。  

### ◆機能
###### ユーザー登録・退会
###### ログイン・ログアウト
###### プロフィール編集
###### レビュー・画像投稿
###### ページング
###### パスワードリマインダー




### ◆使用言語
#### HTML CSS PHP JavaScript MySQL
